# layui-Template

这是一个基于layui框架、参考官方后台模板样式改造的后台管理Template，目前这个还在不断完善中,后期会逐步改进，希望能给大家提供帮助。


# 项目模块结构
    
     |-- layui-Template
       |
       |-- statics
       |    |
       |    |-- css          -- css样式
       |    |-- image        -- 图片
       |    |-- jquery       -- jquery文件
       |    |-- modules      -- 各个模块依赖js
       |    |-- plugins      -- 拓展插件
       |    |
       |-- views
            |
            |- home          -- 子页面
            |- index.html    -- 主页面
